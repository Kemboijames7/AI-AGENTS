// What's happening under the hood
1. Prompt Engineering

The agent's "brain" is an LLM prompted to output structured Thought/Action/Observation JSON on every step.

2. Tool Registry

Each tool is a function with a name, description, and parameters. The LLM picks which to call. The runtime executes it.

3. Memory / Context

Every thought and observation is appended to the context window. The agent always knows its full history.

4. Stopping Condition

Agent stops when it outputs a "Final Answer" action, or when a max-steps limit is hit to prevent infinite loops.
